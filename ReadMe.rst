Readme file for ECCO-GRACE Manuscript
=====================================
Last Update: 02/02/2026 by Nishchitha S Etige

**Nish_100725_ECCO_GRACE_Paper_Figures_V2data_1029.ipynb** contains the code to produce any figure currently included in the ECCO_GRACE manuscript.

- This code uses GRCTellus.JPL.200204_202001.GLO.RL06M.MSCNv02CRI version of GRACE.

- If you need to use other GRACE versions, please read **ReadMe_for_GRACE_versions.txt** on how to obtain the data.
